# Zona Fit GYM
Aplicación de escritorio creada con **Java** y la herramienta de **Swing** para mostrar una interfaz sencilla y amigable al usuario.
Esta aplicación permite el registro y la gestión de los clientes de un gimnasio.

## Funcionalidades
- Lista los clientes registrados
- Agregar un nuevo cliente
- Modificar la información de los clientes ya agregados
- Eliminar un cliente
- Contiene mensajes de confirmación de cada una de las acciones

## Requisitos
- Java 21
- Spring Boot 3.1.2
- MySql

## Instalación
- Clonar el repositorio 
- Abrir el proyecto en tu IDE favorito
- Configurar la base de datos MySQL

## Ejecución
- Seleccionar la clase ZonaFitSwing (src/main/java/gm.zona_fit/ZonaFitSwing)
- Ejecutar la aplicación

## Pruebas Unitarias
- Seleccionar la clase ClienteServicioTest 
- O bien ejecutar cada Test individualmente
